# RuneScape Bot with UI Controller

A bot for RuneScape that completes Tutorial Island and trains on chickens, with a RuneScape-themed UI control panel.

## Installation

1. Install required software:
   - [Node.js 22](https://nodejs.org/)
   - [Java 17](https://adoptium.net/) or later

2. Extract the zip file to a directory of your choice

3. Open a terminal in the extracted directory and run:
```bash
npm install
```

## Running the Bot

1. Start the bot server:
```bash
npm run ui
```

2. Open your web browser and navigate to:
```
http://localhost:8000
```

3. Use the UI to control the bot:
   - Click "Start Bot" to begin
   - Click "Stop Bot" to stop the bot
   - Monitor progress in the chat window
   - Track statistics in the stats panel

## Features

- Tutorial Island automation
- Chicken killing and feather collection
- Random male character generation
- Real-time statistics tracking
- RuneScape-themed UI
- Live activity monitoring

## Important Notes

- Make sure you have a valid RuneScape account before using the bot
- The bot runs locally on your computer
- Port 8000 must be available for the UI server

## Stats Tracking

The UI displays:
- Number of chickens killed
- Feathers collected
- Current location
- Running time

## Troubleshooting

If you encounter issues:
1. Make sure all dependencies are installed
2. Check that port 8000 is not in use
3. Verify your Java and Node.js versions
4. Ensure you have a stable internet connection
